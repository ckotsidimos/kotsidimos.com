# kotsidimos.com

Minimal, long-lived personal authority site for authorship/provenance.
- No services, no funnels, no lead capture.
- Typography-led, static, durable.

## Stack (default)
- Eleventy (11ty) static site generator
- GitHub Pages (via Actions)

## Local dev
```bash
cd site
npm install
npm run dev
```
